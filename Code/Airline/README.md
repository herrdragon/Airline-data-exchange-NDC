## Running the server

1) Clone this git
```
$ git clone ...
```
3) CD into cloned folder
```
$ cd ngds_spring_2019/Airline
```
4) Run
```
$ npm install
```
5) Launch the app from the Terminal:
```
$ node server.js
```
That's it for now.
